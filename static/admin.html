⚠️ 需要更新 Render 環境變數才能永久保存

新增/刪除 ETF 後，必須將以下設定貼到 Render 的環境變數，否則重新部署後會恢復原狀。

更新步驟：

  1. 點上方「複製環境變數值」
  2. 前往 Render Dashboard → etf-tracker-1 → Environment
  3. 找到 ETF_LIST_JSON 環境變數（若無則新增）
  4. 貼上複製的內容 → 點 Save Changes
  5. Render 會自動重新部署，完成後設定永久保留 ✓
